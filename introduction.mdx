# microMLC: Micro Machine Learning Compiler

microMLC is a playground for exploring MLIR and StableHLO compiler passes specifically designed for transformer workloads. It provides a complete pipeline from high-level StableHLO operations to optimized, vectorized MLIR code with focus on tiling, fusion, bufferization, and vectorization optimizations.

## Pipeline Stages

The microMLC compiler pipeline transforms StableHLO operations through multiple optimization stages:

<Steps>
  <Step title="StableHLO to Linalg">
    Convert high-level StableHLO operations into Linalg generic operations with initial tiling
  </Step>
  <Step title="Generic Fusion">
    Fuse producer-consumer chains of Linalg operations to reduce intermediate buffers
  </Step>
  <Step title="Multi-Level Tiling">
    Apply cache-aware tiling at L2 and L1 levels for optimal memory hierarchy utilization
  </Step>
  <Step title="Bufferization">
    Transform tensor operations into memref operations with explicit memory management
  </Step>
  <Step title="Vectorization">
    Generate SIMD vector instructions for parallel computation on modern CPUs
  </Step>
</Steps>

## Toolchain

microMLC builds on top of industry-standard compiler infrastructure:

- **MLIR/LLVM 18.1.8**: Core compiler infrastructure and optimization framework
- **StableHLO v0.13.0**: Stable high-level operations for machine learning
- **Linalg Dialect**: Structured linear algebra operations on tensors and buffers
- **mlir-cpu-runner**: Runtime execution and benchmarking of optimized IR
